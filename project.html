Blood Bank Management System

C++ Development

A Blood Bank Management System is a software program developed in C++ programming language that helps keep track of information about blood donors and donation blood bags as they are moved around in a blood bank. The system is based on the concept of storing blood donor's information and allows the user to perform certain tasks like adding donor's information, listing, and searching for donor's information. The primary goal of the Blood Bank Management System is to manage Blood, Donor, Blood Group, Blood Bank, and Stock data. The system uses classes and other features of C++ and does not use any external files to store the data permanently. There is a simple login system.
